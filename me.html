Reflection

By creating the website myself, I was able to gain a deeper grasp of website design. I learned that creating a website is not as same as designing the landscape projects. Rather than having a big picture in your mind, noticing every element's characteristic also would be a necessary process. They all must have an exact value for its width, height, margin, and padding when utilising HTML and CSS. Additionally, while building a website, I must take into account how it will look on various screen sizes, thus I should always use proportions rather than pixels.


Moreover, in the future, I will reduce the CSS file size and distinguish various elements with class and id.There are certain things I would have done differently if I had known earlier . In terms of CSS, I used to write a separate CSS file for each component of a webpage, such as the navbar, the main body, and the picture portion. However, I discovered that if two CSS files reference the same component, there would be conflicts. This caused me a lot of problems in determining which file I should edit.
